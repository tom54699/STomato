# 常見番茄鐘 App 功能對照（參考自 2025-12-03 研究剪貼）

目的：快速對照本專案已做/未做的常見功能，決定優先級。

## 功能清單與專案狀態
| 功能 | 說明 | 本專案狀態 (React 原型) | 備註/下一步 |
| --- | --- | --- | --- |
| 基礎番茄計時 | 25/5 週期、開始/暫停/重置 | ✅ 已有計時器 | 加入自動休息/長休週期設定 |
| 自訂時長 | 自訂專注/休息長度 | ✅ 圓環可調 5–120 分鐘 | 支援不同預設（25/5、50/10、90/30） |
| 任務/計畫管理 | 任務/計畫清單，估算番茄數 | ✅ StudyPlanner + 今日計畫 | 加入重複計畫、模板、標籤/優先級 |
| 累積番茄追蹤 | 多輪累積完成長計畫 | ✅ targetMinutes/completedMinutes/pomodoroCount | 增加視覺化進度條/徽章觸發 |
| 時間衝突檢測 | 排程衝突提示 | ⚠️ 有提示，需更完整 | 加入跨日/週期性衝突檢查 |
| 數據統計/圖表 | 日/週/月專注時長、分佈 | ⚠️ Insights 初版 | 補足科目/標籤維度、趨勢、目標完成率 |
| 排行榜/社交 | 與好友/全局排名、挑戰 | ❌ 尚無 | 定義 API，最小可先用假資料排行榜 |
| 遊戲化激勵 | 種樹/徽章/成就 | ⚠️ 構想中 | 定義成就規則，與累積邏輯/任務打通 |
| 白噪音/背景音 | 雨聲、咖啡廳等 | ❌ 無 | 評估瀏覽器 Audio + 本地清單 |
| 防打擾/專注模式 | 阻擋分心 App/網站 | ❌ 無 | Web 可先做「全螢幕 + 通知關閉提示」 |
| 通知/提醒 | 開始/結束/計畫提醒 | ❌ 未實作 | 瀏覽器 Notification API，處理授權 |
| 任務整合/日曆 | 行事曆、待辦同步 | ⚠️ 排程頁初版 | 加入月視圖、匯入/匯出 ics/JSON |
| 多裝置同步 | 雲同步 | ❌ 無 | 後端需求：使用者/計時/計畫/排行榜 |
| 資料匯出/備份 | JSON/CSV 匯出、清除 | ❌ 無 | 加入 LocalStorage 匯出/匯入 |
| PWA/離線 | 安裝、離線可用 | ❌ 未設置 | Vite PWA plugin；快取核心路由與資料 |
| 測試/可觀測 | 單元/E2E、日誌 | ❌ 少 | 針對計時/衝突/累積邏輯補測試 |

## 現況 vs 常見功能（速覽，供日後排期參考）
- ✓ 基礎計時、自訂時長、任務/計畫、累積追蹤。
- △ 部分具備：衝突檢測、統計、遊戲化構想。
- ✗ 尚無：通知/提醒、排行榜/社交、白噪音、防打擾、雲同步、匯出/備份、PWA。
- 建議排序：  
  1) 體驗補齊：通知、重複計畫/模板、衝突完善、統計(週/月+標籤)。  
  2) 動機留存：成就/徽章、假排行榜、白噪音。  
  3) 可靠性：匯出/匯入、測試。  
  4) 上線化：PWA、雲同步/後端。  
> 註：此表僅作未來規劃參考，功能是否採納與時程仍待確認。

## 快速優先級建議
- 近期（支撐原型驗證）：通知/提醒、統計完善、重複計畫、假排行榜、成就草案。
- 中期（上線前）：匯出/備份、PWA、白噪音、月視圖/模板化計畫。
- 長期（Flutter + 後端）：雲同步、真排行榜與社群、反分心模式。
